# GEARCALC + STE (NN) - Upload & Render Guide (Greek)

Ο φάκελος αυτός περιέχει **ό,τι χρειάζεται** για:
- UI (`index.html`)
- Python backend (`app/`)
- NN inference files (`NN_model/`)
- Render deploy config (`requirements.txt`, `render.yaml`)

Στόχος: να ανεβάσεις αυτά τα αρχεία στο GitHub repo root και να κάνεις deploy σε Render Web Service.
